Météo

Ce menu permet de générer un fichier GRIB pour la météo.
  1. Après avoir cliqué sur "Météo", un bouton **"Générer GRIB"** apparaît.
  2. Le bot affiche une carte du monde divisée en trois **zones** numérotées (1, 2, 3). Vous devez en sélectionner une.
  3. Choisissez ensuite un **modèle météo** parmi les options disponibles : **ECMWF** ou **GFS**.
  4. Sélectionnez une **date** pour laquelle vous souhaitez le GRIB (format AAAA-MM-JJ).
  5. Le bot vous proposera alors un lien pour le téléchargement du fichier GRIB.

Learn

Ce menu vous permet d'accéder aux modules de formation.
  1. Après avoir cliqué sur "Learn", vous devrez choisir une **thématique** parmi la liste proposée.
  2. Puis, sélectionnez un **niveau** de difficulté pour le cours.
  3. Le cours se lance, affichant son contenu chapitre par chapitre.
  4. À la fin du dernier chapitre, un **QCM** (questionnaire à choix multiples) se lance automatiquement.

Exercices

Testez vos connaissances grâce à des exercices interactifs.
  1. Choisissez un **cours** que vous avez déjà suivi.
  2. Sélectionnez la **difficulté** de l'exercice.
  3. Le bot affichera les **instructions**. Vous pouvez taper vos réponses dans la barre de discussion de Discord.
